# SmartCommission

## What this is
SmartCommission is an incentive compensation management (ICM) platform — a self-contained, world-class solution for designing, calculating, managing, and paying sales commissions and variable compensation.

## Stack
- Next.js App Router + Tailwind CSS v4 + Geist font + lucide-react
- Firebase Auth
- Prisma + PostgreSQL (Cloud SQL)
- GCP: Cloud Run, Cloud SQL, Cloud Build, Secret Manager, Firebase Hosting
